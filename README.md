Repositori ini berisi backend untuk aplikasi review film dan series. Dokumentasi endpoint dipisahkan per modul agar lebih mudah dipakai saat pengembangan frontend dan backend.

## Ringkasan Dokumentasi

| Bagian | Keterangan |
|---|---|
| Backend | API untuk content, publisher, cast, dan review |
| Arsitektur | Express + MySQL |
| Format Response | JSON |
| Dokumentasi Endpoint | Tersedia per modul |

## Modul yang Tersedia

| Modul | Status | Dokumentasi |
|---|---|---|
| Content | Siap | [Buka Dokumentasi](./documentations/content.md) |
| Publisher | Siap | [Buka Dokumentasi](./documentations/publisher.md) |
| Cast | Menyusul | Belum tersedia |
| Review | Menyusul | Belum tersedia |
